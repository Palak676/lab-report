# Laboratory Test Management System

## Overview
A Laboratory Test Android application developed for N.K Diagnostic Laboratory to digitize laboratory semen and urine test reprots.

## Features
- Create new patient semen and urine reports
- Save laboratory reports
- Delete patient records
- Generate printable reports
- User-friendly Android interface

## Technologies Used
- Android Studio
- Java
- HTML
- CSS
- JavaScript
- Android WebView

## Project Domain
Healthcare Management System (HealthTech)
